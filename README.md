# Customer Churn Analysis

## Project Overview
This project performs an end-to-end customer churn analysis using Python and SQLite.

## Tools Used
- Python
- Pandas
- NumPy
- SQLite
- Matplotlib
- Seaborn

## Business KPIs
- Customer Churn Rate
- Average Revenue Per User (ARPU)
- Customer Lifetime Value (CLTV)
- Average Customer Tenure
- Customer Age
- Revenue at Risk
- Complaint Rate
- Escalation Rate
- CSAT Score

## Skills Demonstrated
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Business KPI Calculation
- SQL Queries
- Data Visualization
